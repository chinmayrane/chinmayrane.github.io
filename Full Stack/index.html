Why eXpress?

 eXpress is a wrapper over nodejs for making web applications an easier process for NodeJS devs.

  • Enabling REST routes that are as simple as: app.get(/account/:id, function(req, res){ /* req.params('id') is available */ });
  • Automated HTTP header handling: app.get('/', function(req,res){ res.json({object: 'foo'}); });
  • XSS prevention tools
  • Support for Connect middleware for plugging in synchronous functions to handle different things with requests or responses, such as authentication

What Node.JS Isn't

  • Node.js is not a web server!
    • It's a server-side JavaScript environment, capable of running a web server.
  • Node.js is not a framework!
    • It's a wrapped implementation of the V8 engine that comes with a core set of libraries out of the box.

WHat it really is...

  • Node.js is a hammer, a saw, and a screwdriver.  It's very DIY, but the sky's the limit!
